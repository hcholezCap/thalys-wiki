Liste des boutons et liens

  • Pour le bouton principal, classe à utiliser : "btn-principal" 
    <button type="button" class="btn-principal">
  Bouton principal
</button>
  • Pour le bouton secondaire, classe à utiliser : "btn-secondary" 
    <button type="button" class="btn-secondary">
  Bouton secondaire
</button>
  • Pour le bouton commercial, classe à utiliser : "btn-commercial" 
    <button type="button" class="btn-commercial">
  Bouton commercial
</button>
  • Pour le lien seul, classe à utiliser : "link-only" Lien seul 
    <a href="" class="link-only" alt="">
  Lien seul
  <i class="fa fa-angle-right" aria-hidden="true"><i>
</a>
  • Pour les listes de lien, classe à utiliser : "link-list" Lien liste 
    <a href="" class="link-list" alt="">
  Lien liste
  <i class="fa fa-angle-right" aria-hidden="true"><i>
</a>

Eléments du formulaire

  • Pour les boutons radio, pas besoin de classe spécifique, juste le type "radio" 
      <input type="radio" id="reponse-1" name="question" value="reponse-1">
  <label for="reponse-1">Radio 1></label>
  • Pour les boutons checkbox, pas besoin de classe spécifique, juste le type "checkbox" 
      <input type="checkbox" id="test1">
  <label for="test1">J'accepte les condition générales></label>
  • Pour les listes déroulantes, utiliser la classe "list-select" sur la DIV parente (attention, ca implique un peu de JS, voir le fichier (script.js)
    • 07/14
    • 07/14
     
      <div class="list-select">
    <button type="button">
      07/14
      <i class="fa fa-angle-down" aria-hidden="true"><i>
      <i class="fa fa-angle-up" aria-hidden="true"><i>
    </button>
    <ul class="list-element">
      <li>07/14</li>
      <li>07/14</li>
    </ul>
  </div>
  • Pour les input pas besoin de classe spécifique, mais mettre un div autour avec la classe input ATTENTION IL Y A DU JS POUR CE POINT Input Base 
    <div class="input">
   <label for="" class="placeholder"> Adresse email </label>
   <input type="text" placeholder="Adresse email*">
</div>
    Input OK, ajouter la classe OK 
    <div class="input ok">
   <label for="" class="placeholder"> Adresse email </label>
   <input type="text" placeholder="Adresse email*">
</div>
    Input ERROR, ajouter la classe error

    Champ obligatoire 

    <div class="input error">
   <label for="" class="placeholder"> Adresse email* </label>
   <input type="text" placeholder="Adresse email*">
</div>
